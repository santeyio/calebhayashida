Phantastes Album

What

This is the last album that I produced. It's a concept album based on one of my favorite books by author George MacDonald. I play all the instruments, do all the singing, and mix all the music in Pro Tools. You can check it out on spotify, amazon, itunes, or wherever you listen to music.

Why

  • I wanted to create an album that flows together seamlessly as a whole
  • I wanted to create an album that tells a story. Narrative is something I've come to think is important to give music meaning and this was an experiment in giving music the context of a story.



Spotify Lyrics Data Visualizations

What

This is a fun project I'm working on in django, vue, and the spotify API. It generates bubble charts that describe how often different words appear in the lyrics of songs from your spotify playlists. Check out the code on github and the live version here.

Why

  • I wanted to check out vue. The best (only?) way to learn a new technology is to use it. I now love vue.
  • I wanted to figure out how to run webpack and django side by side -- I'm still trying to figure out exactly how to set that up so that development and deployment is painlessly easy.
  • I wanted to try doing an integration of the spotify API in python.
